# A股全市场数据（每月更新）  
包含全市场每只股票、etf/lof、板块指数，从上市之日起，到今天的每日开、高、收、低、成交量数据，保存到csv中。csv命名为`code+name.csv`的格式，如`000001平安银行.csv`。  
对于*ST股票，*替换为了#。比如`000010#ST美丽.csv`。

## 数据最后更新时间：2026-06-26

## 数据获取实现  
通过`pytdx`库，获取通达信数据。

## 实现逻辑  
先在本地创建txt文件，将获取到的最新股票、基金的代码+名称放在其中，接下来，自动从该txt文件中按照顺序获取代码，然后获取对应股票、基金的每日数据。
